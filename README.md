# AI Problem Solving Assignment

## Problem 1: Tic-Tac-Toe AI

This project implements an AI-based Tic-Tac-Toe game where a human plays against the computer. The AI always makes the best possible move.

### Algorithms Used:

* Minimax Algorithm
* Alpha-Beta Pruning

### Features:

* Interactive web-based interface
* AI vs Player gameplay
* Optimal move selection
* Performance comparison (execution time & nodes explored)

---

## How to Run

1. Open the TicTacToe_AI folder
2. Open index.html in a browser
3. Play the game

## Performance Comparison

- Minimax explores all possible game states, so it is slower.
- Alpha-Beta Pruning reduces the number of nodes explored by eliminating unnecessary branches, making it faster.
- The comparison is based on:
  - Execution time (in milliseconds)
  - Number of nodes explored
